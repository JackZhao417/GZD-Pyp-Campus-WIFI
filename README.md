# 🎓 GZD-Pyp-Campus-WIFI
> 广州职业技术大学校园网多设备验证绕过技术原理，支持同一账号无限制多设备同时在线
---

## ✨ 项目背景
我是真被这个设备验证搞烦了，最多只能一台电脑一个手机登陆同一个账号，以至于我无论如何都要攻破这一难关，在经历了九九八十一难后，我发现我一开始的方向就错了，原来是那样的简单。

## 🚀 核心功能
- 🔓 绕过校园网多设备登录限制，实现多台设备登陆同一个wifi
